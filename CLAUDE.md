## This Repo's Role
This repo is the public face of AEGIS governance doctrine. It contains the Aegis Constitution as a versioned document that external parties can cite, audit, and reference. Versions are tagged and releases are published. This is intentionally public and minimal — no implementation code lives here.

## Related Repos
- aegis — Source doctrine from which this charter is derived
- aegis-docs — Public docs site that renders and hosts this charter
- aegis-initiative — Org home that links to this charter

## Stack
Markdown, Git tags for versioning, GitHub Releases for published versions

## Key Conventions
- Constitution lives in CONSTITUTION.md at repo root
- Versions follow semantic versioning (v1.0.0, v1.1.0, etc.)
- Each version tagged in git and published as a GitHub Release
- Amendments proposed via PR with rationale in PR description
- Branch: main is protected; all changes via PR with 1 required review

Initial draft of the Aegis Constitution v0.1.0
